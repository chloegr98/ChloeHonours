After making our skill, in the designers panel we should drag in the Alexa Skills Kit. In order to configure it, we need to get our Skill ID from Developer Services. So, we should then go to Developer Services, select Alexa and go to Alexa Skills Kit and add a new service.. Once your’ve made the skill, view it’s id and copy it. This is how we integrate our lambda function with developer console. So, we should then go back to AWS and add the skill id to the Alexa Skills kit.
