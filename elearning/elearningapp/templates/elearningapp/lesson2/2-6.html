Slots

A slot is a variable or condition that is set by the user, for example the specific time that the alarm clock is set for. It is used to to narrow down what the user wants. Here is an example:

“Alexa, ask AlarmApp to set an alarm for 4am.”

The slot in this example would be 4am.
